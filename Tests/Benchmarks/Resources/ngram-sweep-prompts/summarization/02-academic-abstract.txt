> **Abstract.** Large language models routinely consume tens of gigabytes of GPU memory during long-context inference, with the key-value cache often dominating peak memory usage as context length grows. Existing post-training quantization methods reduce KV memory by 2-4× but introduce non-trivial perplexity regressions on long-context benchmarks, particularly for instruction-tuned models. We present Cascading Residual Quantization (CRQ), a hybrid scheme that combines coarse 2-bit quantization of the dominant principal components with adaptive 4-bit residual encoding for outlier-prone token positions. CRQ requires no calibration data and can be applied as a drop-in replacement for full-precision KV storage. Across seven instruction-tuned models ranging from 3B to 70B parameters, CRQ achieves a 6.3× memory reduction with under 1% perplexity increase on a battery of long-context tasks (LongBench, RULER, ZeroSCROLLS), outperforming state-of-the-art baselines. We further show that CRQ composes cleanly with KV cache offloading and paged attention, enabling 1M-token contexts on consumer-grade hardware previously limited to 32K.
